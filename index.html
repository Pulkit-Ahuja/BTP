How to use the API?

  1. Generate an API key for the API endpoint <endpoint>
  2. Send the request parameters, i.e., the image and the caption in the request parameters
  3. If you are having difficulty connecting with the API:
    1. Check if the endpoint is correct(case-sensitive)
    2. Check if the value request headers are correct(case-sensitive)
    3. Check if the sent input is indeed a text and image in .jpg or .png format
